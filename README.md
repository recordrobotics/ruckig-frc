# ruckig-frc

Ruckig frc swerve simulation and frc-specific java bindings.

## Build Instructions (simulation app)

1. Run CMake to generate build files:

   ```sh
   cmake -S . -B build
   ```

2. Build the project:

   ```sh
   cmake --build build
   ```

3. Run the executable:

   ```sh
   ./build/ruckig_frc
   ```
